# QuickSmash - A Smalltalk port of the QuickCheck unit test framework

## HOMEPAGE

[http://www.yellosoft.us/quickcheck](http://www.yellosoft.us/quickcheck)

## EXAMPLE

	$ make
	gst-package -t ~/.st package.xml &>/dev/null
	./example.st
	+++ OK, passed 100 tests.
	*** Failed! (-19 )
	+++ OK, passed 100 tests.
